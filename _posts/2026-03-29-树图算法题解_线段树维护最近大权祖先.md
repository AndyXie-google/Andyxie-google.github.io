## 11. 树上加边后到根最短路

> 考试来源：2026年春招-美团-算法策略岗-第一批笔试（第11题）

### 题意简述

给定一棵以 `1` 为根的树，每个点有权值 `x_i`。对每个 `i > 1`：

- 在 `i -> 1` 路径上，找到离 `i` 最近的、且权值严格大于 `x_i` 的祖先 `j`；
- 若存在，则加无向边 `(i, j)`。

求加完所有边后，每个点到根 `1` 的最短距离。

### 解题思路

核心分两步：

1. **求每个点的目标祖先 `jump[i]`**
2. **在增广图上 BFS 求最短路**

难点在第 1 步：

- 直接逐点往上找祖先是 `O(n^2)`，会超时。
- 采用一次 DFS，维护“当前根到当前点路径”的信息。
- 将权值离散化后，用线段树维护：每个权值桶上，当前路径中该权值对应的**最深节点**。
- 对点 `u`，查询区间 `(x_u, +inf)`（离散后是 `[comp[u]+1, m]`）里最深节点，就是离 `u` 最近且权值更大的祖先。

因为 DFS 有回溯，所以每个权值桶还要维护一个栈，进入节点时 push，离开时 pop 并恢复线段树。

### 线段树类设计补充（变量/函数/构建）
[线段树基础](https://oi-wiki.org/ds/seg/)
下面对应 `coding_11.cpp` 里的 `SegTree`：

1. 需要的核心变量

- `int n_`：离散化后的权值个数（线段树下标范围 `[1, n_]`）。
- `vector<SegNode> tree_`：大小通常开 `4 * n_`。
    - `SegNode.depth`：该区间最优候选点的深度。
    - `SegNode.node`：该最优候选点编号。
    - 语义：`tree_[p]` 维护当前区间中“深度最大”的活跃节点（即离当前 DFS 点最近的祖先候选）。

2. 需要的函数

- `SegTree(int n)`：构造函数，完成初始化（可视作 build）。
- `static SegNode better(a, b)`：比较器，返回深度更大的节点。
- `void update(int idx, SegNode val)`：单点更新入口。
- `void update(int p, int l, int r, int idx, SegNode val)`：递归单点更新。
- `SegNode query(int ql, int qr)`：区间查询入口。
- `SegNode query(int p, int l, int r, int ql, int qr)`：递归区间查询。

2.1 函数实现思路总结

- `SegTree(int n)`
    作用是一次性建立容器并初始化。这里不需要单独写 `build`，因为初值统一是 `{-1, 0}`（表示当前路径上没有可用节点），直接把 `tree_` 全部填成默认值即可。
- `better(a, b)`
    这是整棵线段树的“合并规则”。本题目标是找最近祖先，本质上等价于找深度最大的候选点，因此每次都返回 `depth` 更大的那个 `SegNode`。
- `update(int idx, SegNode val)`（入口）
    这一层只负责屏蔽递归细节，把外部的“更新离散值 idx”请求转发到整段区间 `[1, n_]`。
- `update(int p, int l, int r, int idx, SegNode val)`（递归）
    典型单点更新：沿根到叶子路径下钻，命中 `idx` 后覆盖叶子；回溯时用 `better(left, right)` 重新维护祖先区间。这样可以保证每个区间都始终保存该区间最深活跃点。
- `query(int ql, int qr)`（入口）
    做参数边界保护（比如 `ql > qr` 时直接返回空节点），再调用递归查询，避免调用侧重复写判空逻辑。
- `query(int p, int l, int r, int ql, int qr)`（递归）
    典型区间查询：完全覆盖直接返回；部分覆盖就分别查左右子区间并用 `better` 合并。最终得到 `[ql, qr]` 中深度最大的候选节点。

配合 DFS 的含义是：

- `update` 在“进入节点/退出节点”时维护当前根到当前点路径状态；

### 示例说明（按代码变量名）

示例树：

```text
1 (x[1]=5)
├── 2 (x[2]=3)
│   └── 4 (x[4]=4)
└── 3 (x[3]=6)
```

先给出离散化结果（`vals = [3,4,5,6]`）：

- `comp[2]=1`（值 3）
- `comp[4]=2`（值 4）
- `comp[1]=3`（值 5）
- `comp[3]=4`（值 6）

假设 `depth[1]=0`（与当前代码一致，根深度从 0 开始），DFS 主循环中 `seg` 与 `stkByVal` 的变化如下：

1. 进入 `u=1`
- 首次到达，`u==1` 不查询 `jump[1]`。
- 执行 `stkByVal[comp[1]].push_back({depth[1],1})`，并 `seg.update(comp[1], {0,1})`。

2. 进入 `u=2`
- 查询区间 `[comp[2]+1, m] = [2,4]`，对应“权值严格大于 `x[2]=3`”的桶。
- 当前路径上活跃点只有节点 1（值 5），因此 `seg.query(2,4)` 返回节点 1。
- 于是 `jump[2]=1`。
- 然后把节点 2 入路径状态：`seg.update(comp[2], {1,2})`。

3. 进入 `u=4`
- 查询区间 `[comp[4]+1, m] = [3,4]`，对应“权值严格大于 `x[4]=4`”。
- 当前路径活跃点里，节点 2（值 3）不在该区间；节点 1（值 5）在该区间。
- 因此 `seg.query(3,4)` 返回节点 1，得到 `jump[4]=1`。
- 然后节点 4 入路径状态：`seg.update(comp[4], {2,4})`。

4. 退出 `u=4`
- `stkByVal[comp[4]].pop_back()` 后该桶为空，执行 `seg.update(comp[4], {-1,0})` 恢复为空节点。

5. 退出 `u=2`
- 同理清除节点 2 在路径状态中的影响，恢复父节点上下文。

6. 进入 `u=3`
- 查询区间 `[comp[3]+1, m] = [5,4]`，左端大于右端，`SegTree::query` 直接返回空节点。
- 所以 `jump[3]=0`，表示不存在权值严格大于 `x[3]=6` 的祖先。

最终 `jump` 结论：

- `jump[2]=1`
- `jump[4]=1`
- `jump[3]=0`

对应语义：

- 节点 2 最近的“权值严格大于 3”的祖先是 1。
- 节点 4 最近的“权值严格大于 4”的祖先是 1。
- 节点 3 不存在“权值严格大于 6”的祖先。

说明：本代码判定条件是“严格大于”（`>`），即通过查询区间 `[comp[u]+1, m]` 实现，不是“大于等于”。

3. 与本题操作的对应关系

- 进入节点 `u`：更新 `comp[u]` 位置为 `{depth[u], u}`。
- 查询最近大权祖先：`query` 在访问节点 `u` 时，从值域区间 `[comp[u] + 1, m]` 找到最近的`SegNode`大权值祖先
- 回溯退出节点 `u`：恢复 `comp[u]` 到该权值栈中的上一个节点，若无则恢复为 `{-1, 0}`。

### 关键代码（节选）

1) 查询最近的大权值祖先 + 进入路径状态：

```cpp
if (u != 1) {
    SegNode anc = seg.query(comp[u] + 1, m); // deepest ancestor with value > x[u]
    jump[u] = anc.node;
}

int c = comp[u];
stkByVal[c].push_back({depth[u], u});
seg.update(c, {depth[u], u});
```

2) 回溯时恢复路径状态：

```cpp
int c = comp[u];
stkByVal[c].pop_back();
if (stkByVal[c].empty()) {
    seg.update(c, {-1, 0});
} else {
    auto top = stkByVal[c].back();
    seg.update(c, {top.first, top.second});
}
```

3) 加边后统一 BFS：

```cpp
for (int i = 2; i <= n; ++i) {
    if (jump[i] != 0) {
        g[i].push_back(jump[i]);
        g[jump[i]].push_back(i);
    }
}

vector<int> dist(n + 1, -1);
queue<int> q;
dist[1] = 0;
q.push(1);
while (!q.empty()) {
    int u = q.front();
    q.pop();
    for (int v : g[u]) {
        if (dist[v] == -1) {
            dist[v] = dist[u] + 1;
            q.push(v);
        }
    }
}
```
