## 要点速览

- 输出格式题优先确认“是否补零”和“整数是否保留小数位”。
- `list.sort`、`sorted`、切片与对象拷贝语义容易混淆。
- 自定义排序在 Python 3 需通过 `cmp_to_key` 适配。
- `class` 初始化建议把后续会用到的派生属性一次性定义完整。

## 列表排序`list.sort`
```python
points.sort()    # 升序排序
points.sort(key=lambda x: x.dis)    # 按dis升序排序
points.sort(key=lambda x: (x.dis, x.id))    # 优先按dis升序排序, 其次按id升序排序

# functools定制比较函数
from functools import cmp_to_key

def cmp(a, b):
    if a.dis != b.dis:
        return -1 if a.dis < b.dis else 1
    return -1 if a.id < b.id else (1 if a.id > b.id else 0)

points.sort(key=cmp_to_key(cmp))
```
