##### 如何对`map/set`等有序容器进行自定义排序

思路1: 规避新建结构体保存数据; 例如只有2个关键信息, 完全可以使用`set<pair<int, int>`代替自定义结构体

思路2(`2025年秋招-华为-9月17号AI岗-22题`): 完善结构体内容, 重载运算符`<`:
```C++
struct trans {
		string head;
		string tail;
		bool operator < (const trans& other) const {
				if (head != other.head) return head < other.head;
				return tail < other.tail;
		}
};
```

思路3_[STL容器map自定义排序详解](https://zhuanlan.zhihu.com/p/421195917): 新建比较类模板`CMP`, 增加类内比较函数`cmp`, 在定义时增加参数
```C++
class CmpByClass {
public:
    bool operator () (const Stu &a, const Stu &b) {
        if (a.age != b.age) {
            return a.age  < b.age;
        } else if (a.height != b.height)
            return a.height < b.height;
        else
            return true;
    }
};

map<Stu, int, CmpByClass> students;
```

##### 迭代器 iterator 注意事项

参考资料：

- [算竞常用 C++ STL 用法](https://www.bilibili.com/video/BV1L8411y7th)
- [算竞常用 C++ STL 用法(文档)](https://io.zouht.com/154.html)
- [杂项 | 算竞常用 C++ 代码片段](https://io.zouht.com/174.html)

```C++
// 迭代器 iterator 示例1: 为什么 3 没删掉？
vector<int> a{1, 2, 3, 4};
for (auto it = a.begin(); it != a.end(); ++it)
    if (*it == 2 || *it == 3)
        a.erase(it);
// a = [1, 3, 4]
// 答案: 在进行erase操作的时候由于2, 3元素相邻, 迭代器亦相邻, 删除2过后迭代器并不会直接消失, 而是指向3, 因此相当于未进行 *it == 3 的判定

// 迭代器 iterator 示例2: 为啥 RE 了？
vector<int> a{1, 2, 3, 4};
for (auto it = a.begin(); it != a.end(); ++it)
    if (*it == 4)
        a.erase(it);
// 答案: erase 后it迭代器成为"野指针", it == a.end(), 再进行 ++it 后报错
```

总结: 不要使用过期的迭代器 iterator ! 常见的过期场景: 

- erase(), insert()前后; 
- 使用 for(auto it = .begin(); it != .end(); it++) 进行遍历时

// set.insert(x)返回值：pair<set<int>::iterator,bool>

set<int> s;

bool flag = s.insert(x).second;   // 代表插入是否成功的 bool

#### `set/map/vector` 常用查找函数

- `upper_bound(val)`  返回大于等于val的第一个元素的指针
- `lower_bound(val)`  返回小于等于val的第一个元素的指针
- `equal_range(val)`  返回等于val的第一个元素的指针

> `sort(vec.begin(), vec.end(), cmp)` 注意对于vector容器, 传入`sort`的指针最好调用STL的迭代器`it`, 不要直接`sort(vec, vec+len(vec), cmp)`, 特别是对于结构体vec数组

> 标准排序模板: `std::less<int>()`, `std::greater<int>()`, ...

##### `vector`二维数组(`resize`+`fill`)<->`malloc`+`memset`定义二维数组指针

```C++
#include <iostream>
#include <string.h> // memset声明

// 定义n*m矩阵
vector<vector<int>> vec(n);
for(int i = 0; i < n; i++)
{
    vec[i].resize(m);
    fill(vec[i].begin(), vector[i].end(), 0);
}

int** arr = (int**) malloc(sizeof(int*)*n);
for(int i = 0; i < m; i++)
{
    arr[i] = (int*) malloc(sizeof(int)*m);
    memset(arr[i], 0, sizeof(int)*m);
}
```
