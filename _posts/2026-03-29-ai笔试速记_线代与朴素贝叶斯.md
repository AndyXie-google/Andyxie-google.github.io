- sigmoid函数形式: 
  $$
    sigmoid(z) = \frac{1}{1+e^{-z}}
  $$
- 梯度下降法流程熟悉:
```python
### 梯度下降法: 先计算目标函数z, 然后计算通过激活函数(sigmoid)后的映射p, 与原始值y相减得到误差err, 
# err与x进行矩阵运算后求平均即得到权值矩阵w的梯度
# err的均值即为偏置b(一维)的梯度
# 迭代一次: x_k+1 = x_k - lr * grad 
if train_n > 0:
    for _ in range(EPOCHS):
        z = train_x @ w + b
        p = sigmoid(z)
        err = p - train_y

        grad_w = (train_x.T @ err) / train_n
        grad_b = err.mean()

        w -= LEARNING_RATE * grad_w
        b -= LEARNING_RATE * grad_b
```

## 21题（NumPy 线性代数速查）
> 考试来源：2025年秋招-华为-9月17号AI岗（21题）

numpy线性代数相关函数/类内变量:

- 重点函数:
    - `np.ones((r, c))`: 构造全 1 矩阵
    - `np.zeros((r, c))`: 构造全0矩阵
    - `np.zeros_like(x)`: 构造形状和矩阵x一样的全0矩阵
    - `np.triu(mat)`: 取上三角
    - `np.dot(a, b)`: 二维矩阵乘法, 高维可能会有变化
    - `np.cross(x, w1)`向量叉乘
    - `k.T`: 矩阵k的转置
    - `np.linspace(-1,2,5) # 起点为-1，终点为2，取5个点`
    - `np.resize(2,3,4)    # 改变矩阵形状`
    - `np.dtype    # 数组数据类型`

- 常用线性代数函数速查:
    - `A @ B`: 矩阵乘法
    - `np.eye(n)`: 单位矩阵。
    - `np.diag(v)` / `np.diag(A)`: 向量构造对角阵 / 提取对角线。
    - `np.trace(A)`: 矩阵迹。

- `np.linalg` 高频 API:
    - **`np.linalg.inv(A)`: 矩阵求逆(仅方阵且可逆)**
    - **`np.linalg.det(A)`: 行列式**
    - **`np.linalg.matrix_rank(A)`: 矩阵秩**
    - **`np.linalg.norm(x, ord=2)`: 向量/矩阵范数**
    - **`np.linalg.solve(A, b)`: 解线性方程组 `Ax=b`**
    - `np.linalg.lstsq(A, b, rcond=None)`: 最小二乘解(超定/欠定系统)。

- 分解与谱分析:
    - **`np.linalg.eig(A)`: 一般矩阵特征值/特征向量**
    - `np.linalg.eigh(A)`: 实对称(厄米)矩阵特征分解，数值更稳定。
    - `np.linalg.svd(A, full_matrices=False)`: SVD 分解。
    - `np.linalg.qr(A)`: QR 分解。
    - `np.linalg.cholesky(A)`: Cholesky 分解(要求对称正定)。

- 实战注意点:
    - 先检查维度再算: `A(m,n) @ B(n,p) -> (m,p)`。
    - 尽量避免显式求逆: 优先 `solve/lstsq`。
    - 浮点比较不要直接 `==`，用 `np.allclose(a, b)`。
    - 广播容易隐式扩维，必要时先 `reshape` 明确形状。

## 22题（结构体索引与位掩码枚举）
> 考试来源：2025年秋招-华为-9月17号AI岗（22题）

- 当map索引需要设置为结构体时, 如果结构体数量较少, 可以考虑使用`map<pair<string, string>, int>`来进行定义; 如果大于3个, 且需要用到查询操作, 则需要定义比较运算符;
```C++
struct trans {
        string head;
        string tail;
        bool operator<(const trans& other) const {
                if (head != other.head) return head < other.head;
                return tail < other.tail;
        }
};

```

- 核心思路: 位掩码枚举切分点。
    - 长度 `len` 的串有 `len-1` 个可切分位置。
    - 枚举范围应为 `0 .. (1ULL << (len-1)) - 1`
- 切分判定:
    - 扫描位置 `j`，若 `j==len-1` 或该位为 1，则在 `j` 处切一刀。
    - 每段子串用 `dict` 查词典分，失败则该方案无效。
    - 命中相邻词对时再叠加 `bonus` 转移分。
- 易错点:
    - 不要用 `pow(2, len)` 做整数枚举上界(浮点误差风险)。
    - 状态变量 `last_ptr/last_str` 要在“每个掩码方案”内维护，不能每轮 `j` 重置

### 贝叶斯分类器的基本算法流程

给定训练集 `train` 与测试集 `test`，实现二分类 Multinomial Naive Bayes：

- `train` 每行为 `[f1, f2, ..., fm, y]`
- 其中 `fi >= 0` 为词频特征，`y in {0,1}` 为类别标签
- `test` 每行为 `[t1, t2, ..., tm]`

目标：输出每个测试样本的预测标签数组（按输入顺序）。

#### 1) 统计量定义

令：

- `N`：训练样本总数
- `Nc`：类别 `c` 的样本数（`c in {0,1}`）
- `n(c,w)`：类别 `c` 下第 `w` 个词在所有样本中的总词频

对应代码中的核心量通常是：

- `class_count[c] = Nc`
- `class_word_sum[c][w] = n(c,w)`

#### 2) 先验概率

类别先验：

$$
\pi_c = P(c) = \frac{N_c}{N}
$$

解释：反映类别在训练集中的基准占比。

#### 3) 条件概率（拉普拉斯平滑，k=1）

Multinomial NB 中，类条件概率为：

$$
P(w\mid c) = \frac{n(c,w)+1}{\sum_{w'}\bigl(n(c,w')+1\bigr)}
= \frac{n(c,w)+1}{\sum_{w'}n(c,w') + m}
$$

其中 `m` 是词维度数。

平滑的作用：避免某个词在某类别中未出现时出现零概率，导致整条样本分数变为负无穷。

#### 4) 对数后验打分

对测试样本 `x = (x1,...,xm)`，计算：

$$
\log P(c\mid x) \propto \log \pi_c + \sum_{w=1}^{m} x_w \log P(w\mid c)
$$

用对数是为了数值稳定，避免连乘下溢。

#### 5) 参考实现步骤

1. 读取并解析 JSON 输入。
2. 遍历训练集，累计 `class_count` 与 `class_word_sum`。
3. 根据统计量计算 `priors` 与 `prob_wc`（平滑后条件概率）。
4. 对每个测试样本计算两类对数分数并比较。
5. 输出预测数组（JSON）。
