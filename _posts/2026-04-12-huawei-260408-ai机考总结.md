# Huawei_260408 Ai岗位暑期实习第一次机考总结 

## 要点速览

- 题目核心流程: 初始化中心点, 迭代聚类, 计算路径与时间。
- Python/C++ 两个版本逻辑一致, 重点在数据结构与排序实现。
- 易错点集中在: 拷贝语义、排序方式、边界与常量定义。
- 文末提供对话中的关键技术结论, 便于复盘。

## 任务概述
该目录包含 `coding_21.py` 和 `coding_21.cpp` 两个实现，均用于处理如下输入格式：

- 第一行：`k n speed`
- 接下来 `n` 行：每行一个点坐标 `x y`

程序的目标是：
1. 以距离原点最近的 `k` 个点作为初始聚类中心；
2. 对点集执行类似 K-means 的迭代：把每个点分配给最近中心，并更新中心坐标；
3. 最终按照中心点与原点距离排序，并计算一条从原点出发依次经过中心点的总行驶距离与时间。

## Python 实现：`coding_21.py`

### 关键逻辑
- `Point` 类保存 `id`、`x`、`y`、`dis`、`mid`。
- 构造时计算 `dis = sqrt(x^2 + y^2)`，用于按距离排序。
- `points.sort(key=lambda x: (x.dis, x.id))`：按到原点的距离排序，同距离按 `id` 排序。
- 取前 `k` 个点作为初始中心，且使用深拷贝：
  - `k_points = [points[i] for i in range(k)]`
  - 这一步避免之后更新中心时意外修改原始样本点。
- 迭代过程：
  - 每个点按照当前中心点找到最近的 `nearest_k`。
  - 计算每个簇的质心 `(x_new, y_new)`。
  - 计算移动距离 `dis_move`，当其大于 `1e-4` 时继续迭代。
- 最终对 `k_points` 再次排序，输出各中心点和行驶距离，并打印总时间：
  - `floor(total_dis / speed * 3600)`。

### 已修正问题

- 早期实现中 `k_points = points[:k]` 复用对象，导致迭代更新时污染原始数据(问题不大)。

### 备注
1. 深拷贝 vs 浅拷贝
   
2. `class` 的创建和初始化

3. 列表排序list.sort
当前代码中 `functools` 并未使用，但如果需要定制比较函数，Python 3 中应使用：

```python
from functools import cmp_to_key

def cmp(a, b):
    if a.dis != b.dis:
        return -1 if a.dis < b.dis else 1
    return -1 if a.id < b.id else (1 if a.id > b.id else 0)

points.sort(key=cmp_to_key(cmp))
```

`functools` 是 Python 标准库模块，可直接 `import functools` 使用，但它不是内置模块。

## C++ 实现：`coding_21.cpp`

### 关键逻辑
- 使用 `struct Point` 保存 `id`、`x`、`y`、`dis`、`mid`。
- 读入所有点之后按 `dis` 排序，然后取前 `k` 个点作为初始中心。
- 采用类似的迭代方式，将每个点分配给最近中心，并更新簇中心坐标。
- 最终按 `dis` 排序中心点，计算从原点按顺序经过所有中心点的距离总和与时间。

### 当前实现中的注意点
- `#define inf 2^63-1` 是错误写法，已修正为：
  - `#define inf 9223372036854775807LL`
  - 因为 `^` 在 C++ 中是按位异或，不是幂运算。

- 建议使用`const long long inf = pow(2, 63) - 1替代`

## 对话要点总结

- Python 3 的 `list.sort()` / `sorted()` 不支持直接使用 `cmp` 参数。
- 需要自定义比较函数时，应使用 `functools.cmp_to_key()`。
- `functools` 是 Python 的标准库模块，可以直接导入。
- 对于这份题目，直接使用 `key=lambda x: (x.dis, x.id)` 可以达到按距离再按 `id` 排序的效果。

## 结论

- 本题实现重点并非复杂模型, 而是工程细节是否严谨。
- 同一算法在 Python 与 C++ 中都应优先保证排序规则、对象语义和循环边界正确。
- 若先确保正确性, 再逐步优化性能, 更容易稳定拿分。
